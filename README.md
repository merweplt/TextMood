# TextMood

**TextMood**, yapay zekanın temel mantığını basit bir şekilde simüle eden bir projedir.  
Bu proje, kullanıcıdan aldığı metinleri analiz ederek **pozitif, negatif veya neutral duygu tahmini** yapar.  

## Özellikler
- Basit AI simülasyonu
- Duygu skoruna göre tahmin
- Emoji ve kısa yorum ile görsel çıktı


## Kullanım
1. Python ortamında çalıştırın:
```bash
python main.py

Program sizden cümle alacak. Örnekler:
Cümle yazın: Bugün çok mutluyum!
AI Tahmini: Pozitif 😊 - Harika hissediyorsun!

Cümle yazın: Üzgün hissediyorum
AI Tahmini: Negatif 😞 - Moralin bozuk gibi...

Cümle yazın: Merhaba
AI Tahmini: Neutral 😐 - Normal bir gün
Çıkmak için 'q' yazın.
